DOCUMENTACION API IFA

/docs

Método: GET

Resumen: Obtener documentación de la API

/orden

Método: POST

Descripción: Permite crear una nueva orden en el sistema.

Request headers:

{
        "authorization": "token otorgado por IFA"
      }

Request Body:


    {
        "CardCode": "C000004",
        "DocDate": "2024-08-22",
        "DocDueDate": "2024-08-22",
        "DocumentLines": [
          {
            "ItemCode": "101-001-001",
            "Quantity": 1,
            "TaxCode": "IVA",
            "UnitPrice": 30,
            "WarehouseCode": "100-12"
          }
        ]
      }

Respuesta Ejemplo:

{
    "message": "Orden grabada con éxito",
    "orderNumber": "421829",
    "status": 204,
    "statusText": "No Content"
}

/entrega

Método: POST

Descripción: Permite crear una nueva entrega de venta en el sistema.

Notas: Cada entrega creada debe tener una orden de venta origen en el campo BaseEntry de las lineas.

Request Body:

{
    "CardCode": "C000004",
    "DocDate": "2024-08-22",
    "DocDueDate": "2024-08-22",
    "DocumentLines": [
      {
        "BaseType": 17,
        "BaseEntry": 421827,
        "BaseLine": 0,
        "LineNum" : "0",
        "ItemCode": "101-001-001",
        "Quantity": "1",
        "TaxCode": "IVA",
        "UnitPrice": "30",
        "WarehouseCode": "100-12",
        "BatchNumbers" : [ { 
            "BaseLineNumber" : "0",
            "BatchNumber" : "92351", 
            "Quantity" : "1", 
            "ItemCode" : "101-001-001" } ]
      }
    ]
  }

Respuesta Ejemplo:

{
    "message": "Entrega grabada con éxito",
    "orderNumber": "17998",
    "status": 204,
    "statusText": "No Content"
}
